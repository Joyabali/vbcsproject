Please Note

When using this template you need to select the correct theme version for the version of Visual Builder that you are using. This can be changed via the Settings screen for the WebApp.

The versions to use are as follows:

  • ApplicationsCloudUI/1.1.0/web/ApplicationsCloudUI.min.css - For Oracle Visual Builder *19.1.n* and below
  • ApplicationsCloudUI/2.0.0/web/ApplicationsCloudUI.min.css - For Oracle Visual Builder *19.3.n*
  • ApplicationsCloudUI/3.0.1/web/ApplicationsCloudUI.min.css - For Oracle Visual Builder *19.4.n*
Also note that if your Visual Builder instance is upgraded you may need to switch to, or import, a new version of the theme to allow applications that you have created with this template to continue to work correctly. You should read the "What's New" documentation when you are upgraded and this will advise you if changes are required to the theme used in this template
